# Contributing to Awesome SIE Exam Prep

Thank you for your interest in contributing! This list aims to help people pass the SIE exam with free and quality resources.

## Guidelines

### What We Accept

- Free resources (study guides, practice tests, videos)
- Official resources from FINRA, SEC, etc.
- Quality paid resources (for reference section only)
- Helpful Reddit threads or community discussions
- Mobile apps with free tiers
- YouTube channels with educational content

### What We Don't Accept

- Resources with affiliate links
- Spam or low-quality content
- Resources behind paywalls with no free tier
- Outdated resources (must be relevant for current exam)
- Self-promotion without genuine value

## How to Contribute

1. **Fork** this repository
2. **Add** your resource to the appropriate section
3. **Format** your addition like this:
   ```markdown
   - [Resource Name](https://example.com) - Brief description of what it offers
   ```
4. **Submit** a pull request with a clear description

## Resource Format

Please include:
- Name of the resource
- Direct link (no redirects or affiliate tracking)
- Brief description (1 sentence)
- Any special notes (free tier limits, etc.)

## Questions?

Open an issue if you have questions or suggestions for improving this list.
